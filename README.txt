Собрано с помощью бандлера Vite. Сборка проекта для запуска в браузере - index.html в папке dist, остальное - исходный код.

Общее время (включая перерывы) - 3 дня с момента получения задания, 2 дня с начала работы.

Чистое время - 8 часов.

Особых сложностей в задаче не нашёл, основное время заняли нарезка макета и pixel-perfect вёрстка.


Относительно принятых решений:

Кнопка перемещения отключается в процессе анимации и после 5 реализованных переходов.

Лента в блоке снизу наполняется данными из списка друзей.

Кнопки ленты отключаются при перемотке в начало/конец.

Аватары male и female, указанные в data, но не предоставленные в макете, изобразил схематично (цветовая полоса).
